# Grafterr Landing Page

## Tech Stack

* React 18
* CSS3
* JavaScript (ES6)

## Features

* Dynamic content from JSON (mock API)
* Responsive carousel (1/2/3 items)
* Loading skeletons
* Error handling with retry
* Clean component-based architecture

## Project Structure

* components/ui → reusable components
* hooks → custom hooks
* services → API layer
* sections → page sections

## Setup
